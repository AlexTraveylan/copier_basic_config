# Template for configure python project

## Features

- config .vscode
- config ruff
- config mypy
- config pytest

## Usage

```bash
uv tool install copier
```

Then

```bash
copier copy https://github.com/AlexTraveylan/copier_basic_config.git path/to/destination
```
